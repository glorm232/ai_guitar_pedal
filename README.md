# AI Guitar Pedal
An ESP32-S3-based real-time guitar effects pedal, this project starts with classic DSP, then in the future move on to intergrating neural-network inference.

## Structure
- hardware/
- firmware/
- dsp_experiments/
- neural_network_models/
- test_recordings/
- documentation/
- tools/

## First milestone

Clean real-time audio pass-through on ESP32-S3 with acceptable latency(< or = 10ms).
